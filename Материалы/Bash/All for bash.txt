~ - домашний каталог

. - относительный путь текущего рабочего каталога

.. - Подняться вверх по  директории

&& - 'and' 'и'

|| - "или" 'or'
 
ls - Вывод список файлов в директории
rm - удаление файлов
cd - изменяет путь работы с файлами
cat - открытие файла(txt)
scp (Secure CoPy) — утилита для безопасного копирования данных между Linux-системами по протоколу SSH.
less - пейджер,удобный просмотр файлов и поиск текста в нем(/ - поиск)
head - отобразит первые 10 строк файла
tail - последние строки файла
grep - поиск чего то конкретного(grep [слово которое нужно] [файл](Флаг -v убирает слово из поиска)
cp - копирование файлы и каталоги
touch - создание файлов
mkdir - создание директорий
rmdir - удаление директорий (каталога)
mv - переименование файлов/директорий(Перенос)(move)
alias - псевдонимы команд.Т.е - можно сделать так: alias nuke='rm -rf /' . Удаление всего командой nuke.(Подробнее в файле alias)
ps - Вывод процессов(Без флагов,он выведет процессы терминала в котором вы находитесь,так же текущего пользователя),так же при вводе команды ps,вы получаете информацию,которая не обновляется.То есть,информация собралась и отправилась вам.

nmcli - Текстовое редактор для управления сетью
nmtui - Текстово-Графический редактор для управления сетью



Как лучше использовать те или иные команды
---------------------------------------------------------
cp{
	Лучше копировать так: cp -a /source/* /target
	
	Примеры использования:
	cp /var/log/apt/history.log ~/newhistory.log
	cp /var/log/apt/history.log newhistory.log

	Копирование нескольких файлов:
	cp /var/log/apt/history.log /var/log/dpkg.log <Директория>(Директория может быть создана)

	Копирование каталога:
	cp /var/log/apt apt_log - Копирование каталога путем переноса в директорию apt_log(Если директории нету,он автоматически ее создаст)

	При повторном копировании,внутри директории будут и скопированные файлы/директории, и в нашем случае apt(Копирует последнюю директорию /var/log/apt , то есть apt)

	Флаги:
	-i -interactive(Спросит,хотите ли вы перезаписать файл?)
	-v (Показывает совершенные действия,перезапись,перемещение и тд)
	
}

mkdir{
	mkdir -p d/e/f - создаст сначало d,далее в d создаст e,после в e создаст f.(-p - parent)

	mkdir -v new - подробный вывод информации(-v)(Об успешном создание директории)
		output: mkdir: created directory 'new'
}

rm{
	-f - форсировать(Удаляет без вопросов)
	-r - рекурсивно(Удаление всего в каталоге)
}
ps{
	Для полного вывода всех процессов,используются флаги aux.Где:
	a - Расширенный вывод информации о процессах(PID,TTY,TIME,STAT,CMD)
	u - Вывод информации о запущенном процессе от пользователя(USER)
	x - Вывод всех процессов не подключенные к терминалу.
}

resolve - Преобразование Доменного имени в ip адрес(Если проблема в нем,то пробуем изменить парамент Dns сервера)
