root (GID=0):
Группа суперпользователя. Только root принадлежит к этой группе.
sudo (GID=27):
Группа, предоставляющая право использовать команду sudo для выполнения команд от имени root.
adm (GID=4):
Группа администраторов. Пользователи этой группы могут читать системные журналы и конфигурационные файлы.
dialout (GID=20):
Группа для доступа к модемам и устройствам последовательного порта.
cdrom (GID=24):
Группа для доступа к CD-ROM устройствам.
floppy (GID=25):
Группа для доступа к дискетам (реже используется в современных системах).
tape (GID=26):
Группа для доступа к магнитофонным устройствам.
dip (GID=30):
Группа для доступа к PPP (Point-to-Point Protocol) для работы с сетевыми соединениями.
plugdev (GID=46):
Группа для доступа к USB-устройствам (например, флешкам).
lpadmin (GID=116):
Группа для управления принтерами и печатью.
audio (GID=29):
Группа для доступа к аудиоустройствам.
video (GID=44):
Группа для доступа к видеокартам и другим видеоустройствам.
netdev (GID=120):
Группа для управления сетевыми интерфейсами.
sambashare (GID=126):
Группа для доступа к файлам через Samba.
docker (GID=999):
Группа для управления Docker контейнерами.
